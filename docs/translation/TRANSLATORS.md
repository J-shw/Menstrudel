Thank you for helping translate Menstrudel! All our translations are managed on Crowdin, which makes it easy for anyone to help.

**[Go to the Menstrudel Crowdin Project](https://crowdin.com/project/menstrudel)**

### How to Contribute
1.  Sign up or log in to Crowdin.
2.  Join our project.
3.  Select the language you want to help with.
4.  Start translating! You can add new translations or suggest improvements for existing ones.

### How to Request a New Language
If your language isn't listed, please request it directly on the Crowdin project page. Once it's approved by us, you and others can start translating.

You do not need a GitHub account or any coding knowledge to contribute translations. All your work is done on the Crowdin website, and we handle all the code.
