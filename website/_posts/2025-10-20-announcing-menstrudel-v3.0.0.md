### What's New
* **Biometrics Support** You can now use your device's biometrics (fingerprint/face recognition) to access the app.
* **Persistent Tampon Reminder Button** Added an option for a persistent button to set a notification reminder for tampons.
* **New Period Reminder Button** Implemented a new floating action button (FAB) for period reminders.
* **New About Section** The app now includes a dedicated "About" section.
* **Tampon Reminder Countdown** Added a countdown screen to the tampon reminder.
* **Export Pill Data** You can now export your pill log data.
* **Export Period Data** You can now export your period log data.
* **Data Import** Added the ability to import your data.
* **Edit Past Pill States** You can now edit the state (taken/skipped) of past pills.
* **Multi-Pill Pack Support** The app now supports multiple pill packs.
* **Italian Translations** Thanks to the hard work of a dedicated user, the app is now completely translated to Italian.

### Bug Fixes
* Fixed a crash in the calendar view that occurred with the initial log of the month.
* Fixed incorrect snackbar text when marking a pill as skipped.
* Fixed an issue where future dates were not clickable.
* Fixed a bug where the Wear OS app was not sending data.
* Fixed an issue where the tampon reminder button was not showing as running.
* Fixed a bug related to importing data from a cloud service while offline.
* Fixed an issue where marking a pill as taken did not cancel the reminder.
* Fixed a bug that disabled pill actions when the pack start date was in the future.
* Fixed an issue where placebo pills were not showing the skipped state.
